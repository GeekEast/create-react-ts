## Create React App From Scratch
### Features
- Webpack
- React
- Typescript
- ESLint
- Prettier
- Husky
- Jest
- Enzyme
- React Hot Loader
```javascript
...
import { hot } from 'react-hot-loader';
@hot(module)
class App extends ...
```

### Install Dependency
```sh
yarn
```

### Start
```sh
yarn start
```

### Build
```sh
yarn build
```
