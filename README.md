## Create React App From Scratch
### Features
- Webpack
- React
- Typescript
- ESLint
- Prettier
- Husky
- Jest
- Enzyme
- React Hot Loader
```javascript
...
import { hot } from 'react-hot-loader';
@hot(module)
class App extends ...
```

### Prepare
- **Step 1**: `git clone git@github.com:GeekEast/create-react-ts.git project_folder_name` 
- **Step 2**: `cd project_folder_name && rm -rf .git`

### Install 
```sh
yarn
```

### Start
```sh
yarn start
```

### Build
```sh
yarn build
```
